Ejercicio 1

Realizar un programa en el cual se solicite la edad de una persona. Si la persona es mayor o igual a 18 años, deberá mostrar en pantalla: Usted es mayor de edad.


                        package com.sofka.taller2;

                        public class Ejercicio1T2 {

                            private Scanner input;

                                input = new Scanner (System.in);
                                
                                askOld();
                                old = captureDataInt();
                                
                                if (old >= 18) {
                                    System.out.println("Usted es mayor de edad");
                                }
                            }
                            
                            private void askOld() {System.out.println("¿Cuántos años tienes?");}
                            
                            private int captureDataInt() {
                                return input.nextInt();
                            }
                        }

Ejercicio 3

Realizar un programa en el cual se solicite el nombre, apellidos y edad de la persona. Si la persona es mayor o igual a 18 años, entonces se deberá imprimir en pantalla [Nombre completo] usted es mayor de edad, por lo tanto puede entrar a la fiesta. Si la edad de la persona es menor que 18 años, entonces, deberá imprimirse el siguiente mensaje: [Nombre completo] usted es menor de edad, por lo tanto, no puede entrar a la fiesta, por favor devuélvase a su casa.


                        package com.sofka.taller2;

                        import java.util.Scanner;

                        public class Ejercicio3T2 {

                            private String name;
                            private String surname;
                            private int old;
                            private Scanner input;
                            
                            public void Ejercicio3T2() {
                                
                                input = new Scanner (System.in);
                                
                                askName();
                                name = captureDataString();
                                
                                askSurname();
                                surname = captureDataString();
                                
                                askOld();
                                old = captureDataInt();
                                
                                if (old >= 18) {
                                    System.out.println(name + " " + surname + ", usted es mayor de edad, por lo tanto puede entrar a la fiesta");
                                } else {
                                    System.out.println(name + " " + surname + ", usted es menor de edad, por lo tanto no puede entrar a la fiesta, por favor devuélvase a su casa.");
                                }
                                
                            }
                            
                            private void askName() {System.out.println("¿Cuál es tu nombre?");}
                            private void askSurname() {System.out.println("¿Cuáles son tus apellidos?");}
                            private void askOld() {System.out.println("¿Qué edad tienes?");}
                            
                            private String captureDataString() {
                                return input.nextLine();
                            }
                            
                            private int captureDataInt() {
                                return input.nextInt();
                            }
                        }
